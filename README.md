# Foot-Operated Recorder (Volunteer Project)

Goal: design and iterate an assistive device so a little girl in Japan (no arms) can play a soprano recorder using her feet. This is a volunteer, non-medical project led by KOEIGO STEAM & Language School.

## Design constraints (evolving)
- Enable covering/activating recorder tone holes via toes (precision & reach).
- Comfortable, secure, adjustable; easy to clean.
- No permanent modification to recorder if possible.
- Prioritize low-cost, easy-to-print, parametric parts.

## Ethics & privacy
Do not upload personally identifying images of the child. Use anonymized photos or diagrams. See `docs/safety-and-ethics.md`.
