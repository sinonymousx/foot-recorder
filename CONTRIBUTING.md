## Quick start for contributors
1) Open an Issue with your idea (sketches welcome).
2) Create a branch or fork: `feature/<concept>-v0`.
3) Add files in `designs/<concept>/`:
   - `CAD/` (STEP or native; parametric welcome)
   - `STL/`
   - `BOM.md`
   - `build-notes.md` (print settings, assembly, risks)
4) Open a Pull Request; add photos/diagrams (no PII).
