# Contributing

1. Create a branch: `feature/your-concept`.
2. Put CAD (.step or native + optional .scad/.fcstd for parametric), STLs, BOM, and build-notes in `designs/<your-concept>/`.
3. Open a PR with:
   - Purpose and mechanism explanation
   - Print settings & materials
   - Assembly steps & photos/diagrams (no PII)
   - Risks and test results (comfort, stability, hygiene)

Label the PR with: `concept`, `prototype`, `needs-testing`, or `ready-for-tryout`.

Please keep discussion kind and focused. See `CODE_OF_CONDUCT.md`.
